# 恋爱占有欲指数测试网站 - 使用说明

## 快速开始

### 1. 运行网站

由于浏览器安全限制，不能直接打开HTML文件，需要使用本地服务器运行：

**方法一：使用Python（推荐）**
```bash
# 在项目目录下运行
python -m http.server 8000
```
然后访问：http://localhost:8000

**方法二：使用Node.js**
```bash
# 需要先安装http-server
npm install -g http-server

# 在项目目录下运行
http-server -p 8000
```

**方法三：使用VS Code**
- 安装 "Live Server" 插件
- 右键点击 `index.html`，选择 "Open with Live Server"

### 2. 测试流程

1. **首页**：选择"给自己测"或"为恋人测"
2. **授权码验证**：输入10位授权码（目前为模拟验证，任意10位字母数字组合即可）
3. **知情同意**：阅读并勾选同意
4. **个人信息**：填写基本信息（必填：年龄段、性别、恋爱状态）
5. **开始答题**：依次回答40道题目，支持前进/后退
6. **查看报告**：答题完成后自动生成报告，包含RPI指数和详细分析

### 3. 查看测试记录

在首页访问 `index.html#records` 或点击相关链接，可以查看历史测试记录。

## 功能说明

### 授权码系统
- 目前为模拟验证，格式正确的10位字母数字组合即可通过
- 实际项目中需要连接后端API进行真实验证

### 数据存储
- 所有数据存储在浏览器LocalStorage
- 清除浏览器缓存会丢失所有数据
- 支持答题进度自动保存和恢复

### RPI计算
- 基于4个维度（控制欲望、嫉妒强度、情感依赖、关系不安）
- 使用标准化z分数和百分位数转换
- 参考10000+样本常模数据

### 报告功能
- RPI指数总览（0-100分）
- 维度拆解分析
- 可视化图表（柱状图、雷达图）
- 个性化专家建议
- 沟通技巧指南

## 注意事项

1. **浏览器兼容性**：建议使用Chrome、Firefox、Edge等现代浏览器
2. **数据安全**：所有数据仅存储在本地，不会上传到服务器
3. **授权码验证**：当前为模拟验证，实际部署需要连接后端
4. **导出功能**：PDF和图片导出功能已预留接口，需要集成相应库

## 技术栈

- HTML5
- CSS3（响应式设计）
- JavaScript (ES6+)
- Canvas API（图表绘制）
- LocalStorage API（本地存储）

## 项目文件说明

- `index.html` - 首页
- `consent.html` - 知情同意页面
- `info.html` - 个人信息填写页面
- `test.html` - 答题页面
- `report.html` - 报告页面
- `questionBank.json` - 题库数据
- `css/` - 样式文件目录
- `js/` - JavaScript脚本目录

## 开发与定制

### 修改常模数据
编辑 `js/config.js` 中的 `normData` 对象

### 修改题库
编辑 `questionBank.json` 文件，或使用 `parse_questions_v2.py` 重新解析docx文件

### 添加后端API
在 `js/config.js` 和 `js/utils.js` 中修改相关函数，连接真实后端API
